# Al-Shifa Medical Clinic

Modern responsive medical clinic website built using HTML, CSS and JavaScript.

## Features
- Responsive Design
- Arabic/English Support
- Dark Mode
- Doctors Section
- Appointment UI
- Smooth Animations

## Technologies
- HTML5
- CSS3
- JavaScript

## Note
This project is frontend only and does not include backend functionality.
